# 3DS ARM9 Interrupts


**ARM9 Interrupts (ARM9 only)**
Mostly same as GBA/NDS/DSi, but without IME (only CPSR irq enable), and
without IE2/IF2, and with changed IRQ bits:
10001000h - IRQ_IE - ARM9 Interrupt Enable
10001004h - IRQ_IF - ARM9 Interrupt Flags
