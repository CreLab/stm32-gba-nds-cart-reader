# DSi SD/MMC Firmware Log Files


**FAT16:\\sys\\log\\shop.log (32 bytes)**

```
+-----------------------------------------------------------------------+
|       0000h 20h   Zerofilled                                          |
+-----------------------------------------------------------------------+
```

Unknown if this file can contain anything else.
