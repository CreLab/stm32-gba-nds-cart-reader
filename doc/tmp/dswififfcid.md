# DS Wifi FFC ID


**Wifi FCC ID**
The wifi hardware has been registered by Mitsumi, Hon Hai, and Nintendo
themselves:

```
+-----------------------------------------------------------------------+
|       https://fccid.io/BKE - Nintendo                                 |
|       https://fccid.io/EW4 - Mitsumi                                  |
|       https://fccid.io/MCL - Hon Hai (Foxconn)                        |
+-----------------------------------------------------------------------+
```

The above webpages include some unrelated stuff (like bluetooth), and
some duplicate entries. The wifi/console related entries are:

The pages include test reports, photos (including some prototypes), and
most interestingly, the Mitsumi \"Label location\" pages are includin
pinout & signal names for most wifi boards.

Functionally same as DSi, but not pin-compatible, and with two UART pins
(instead of the NC pins).

GBA-Micro: device has two channels only: 2426.248MHz and 2456.576MHz.
