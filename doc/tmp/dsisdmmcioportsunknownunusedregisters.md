# DSi SD/MMC I/O Ports: Unknown/Unused Registers


Below registers don\'t seem to be used by existing software\...

**40048F2h/4004AF2h - Can be 0003h**

```
+-----------------------------------------------------------------------+
|       15-2   Always zero                                              |
|       1-0    U                                                        |
| nknown (0..3)                                                   (R/W) |
+-----------------------------------------------------------------------+
```


**40048F4h/4004AF4h - Can be 0770h**

```
+-----------------------------------------------------------------------+
|       15-11  Always zero                                              |
|       10-8   U                                                        |
| nknown (0..7)                                                   (R/W) |
|       7      Always zero                                              |
|       6-4    U                                                        |
| nknown (0..7)                                                   (R/W) |
|       3-0    Always zero                                              |
+-----------------------------------------------------------------------+
```
