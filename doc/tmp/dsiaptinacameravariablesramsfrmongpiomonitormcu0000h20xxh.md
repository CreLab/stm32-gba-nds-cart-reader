# DSi Aptina Camera Variables: RAM/SFR/MON (GPIO/Monitor) (MCU:0000h-20xxh)


**Internal RAM (MCU:0000h..0xxxh)**
Internal RAM is reserved for whatever internal purposes (probably
including for storing the \'logical variables\' at MCU:2xxxh at som
physical memory locations at MCU:0xxxh). However, some of those
undocumented reserved RAM cells are manipulated by DSi games:

```
+-----------------------------------------------------------------------+
|       02F0h   2  UNDOC_RAM_02F0   (set to 0000h by DSi games)         |
|       02F2h   2  UNDOC_RAM_02F2   (set to 0210h by DSi games)         |
|       02F4h   2  UNDOC_RAM_02F4   (set to 001Ah by DSi games)         |
+-----------------------------------------------------------------------+
```

Exact RAM Size is unknown (around 2Kbyte or so)? Some Aptina chips do
also contain some sort of User RAM (at 0800h or so) for unknown purpose
(just general purpose storage maybe). Unknown if the DSi chips are
having any such User RAM.
