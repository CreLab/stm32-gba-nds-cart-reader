**Unused command numbers PICA(0000h..000Fh, 0020h..003Fh)**
Registers at 10401000h..1040103Fh and 10401080h..104010FFh are writeable
by ARM11 only, not via PICA command numbers 0000h..000Fh, 0020h..003Fh.

**Unused command numbers PICA(0400h..FFFFh)**
The PICA command numbers in CMDBUF are said to be 16bit wide, however,
values 0400h..FFFFh seem to be unused (and there are no I/O addresses
for them; accessing 10402000h and up triggers data abort).
