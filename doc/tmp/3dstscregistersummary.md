# 3DS TSC, Register Summary


**Page Selection**

```
+-----------------------------------------------------------------------+
- |       TSC[x                                                           |
| xh:00h]=page    ;Page (each TSC SPI bus probably has own page+index?) |
+-----------------------------------------------------------------------+
```


**Page 00h-01h (DSi Registers)**

```
+-----------------------------------------------------------------------+
- |       TSC[00h:02h]=read    ;DSi Undocumented status (reserved bits)   |
|                                                                       |
- |     TSC[00h:03h]=read    ;DSi Overtemperature OT Flag (reserved bits) |
- |       TSC[00h:0Bh]=87h     ;DSi DAC NDAC Value                        |
- |       TSC[                                                            |
| 00h:39h]=66h     ;DSi ADC DC Measurement 1 (reset=00h, ORed with 66h) |
- |       TSC[                                                            |
| 00h:3Fh]=D4h     ;DSi DAC Data-Path Setup  (reset=D4h, ORed with C0h) |
- |       TSC[00h:40h]=00h     ;DSi DAC Volume Control                    |
- |       TSC[00h:                                                        |
- | 41h]=FDh     ;DSi DAC Left Volume Control  ;\aka 3DS     ;HWCAL[2E4h] 
- |       TSC[00h:                                                        |
- | 42h]=FDh     ;DSi DAC Right Volume Control ;/ShutterVol0 ;HWCAL[2E4h] |
|                                                                       |
- |     TSC[00h:51h]=00h/80h ;DSi Microphone Mute/Unmute ;ADC Digital Mic |
|                                                                       |
- |  TSC[00h:52h]=80h/00h ;DSi Microphone Mute/Unmute ;ADC Digital Volume |
- |       TSC[01h:2Eh]=03h     ;DSi Microphone MIC BIAS                   |
- |       TSC[01h:2Fh]=2Bh     ;DSi Microphone MIC PGA                    |
- |       TSC[01h                                                         |
| :30h]=40h     ;DSi P-Terminal ADC Channel Fine-Gain Input (reset=40h) |
- |       TSC[01h                                                         |
| :31h]=40h     ;DSi M-Terminal ADC Input Selection         (reset=40h) |
+-----------------------------------------------------------------------+
```

The 3DS does usually access only the registers mentioned above (but
there are many more DSi-style registers in page 00h,01h,03h; see DSi
chapter for details).

```

The above coefficient RAM pages exists on DSi, too. However, the DSi is
usually initializing only those in page 04h.
Caution: The various 16bit Filter values in the HWCAL and config files
are stored in LITTLE-ENDIAN, but the TSC chip wants them in BIG-ENDIAN,
so one must swap each 2 bytes.
Unknown how the 3DS is using the extra coefficients\... does it us
miniDSP instructions for that?
Unknown what the duplicated entries are for\... maybe left/righ
channels?

**Page 64h (100 decimal) (3DS Sound/Microphone Config)**

```
+-----------------------------------------------------------------------+

- |       TSC[64h:26h]=read    ;3DS status, wait for bit3,7               |

- |       TSC[64h:44h]         ;unused, but nonzero                       |
