```
+------------------------------------------------------------------------------------+
|       DDI0424D_dma330_r1p2_trm        - DMA controller (also covers r0p0 and r1p1) |
|       IHI0022D amba axi_specification - Memory (ARCACHE, ARPROT etc.)              |
+------------------------------------------------------------------------------------+
```
