The 200h-byte Public Key for the \'Root\' signature is reportedl
somewhere in NATIVE_FIRM (unknown where though, knowing the first some
bytes would help) (also unknown if it is the same \'Root\' key as o
DSi).
The DSi has a similar file called cert.sys:
- [DSi SD/MMC Firmware dev.kp and cert.sys CertificateFiles](./dsisdmmcfirmwaredevkpandcertsyscertificatefiles.md)
The DIFF container uses the standard DISA/DIFF format
- [3DS Files - Savedata DISA and DIFF](./3dsfilessavedatadisaanddiff.md)
However, the DIFF in certs.db does contain raw data (instead of the
SAVE/BDRI/VSXE filesystems that are usually found in DISA/DIFF
containers).
